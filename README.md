# Key Code Viewer

A small interactive JavaScript project that displays the key pressed by the user along with its key code value in real time.

## Features
- Real-time keyboard input detection
- Displays pressed key value
- Shows corresponding key code
- Instant DOM update
- Supports letters, numbers, and special keys

## Tech Stack
- HTML
- CSS
- JavaScript
